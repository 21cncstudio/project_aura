# Home Assistant Dashboard

This folder contains a ready-to-use dashboard YAML.

## How to use
1) In Home Assistant, open your dashboard and switch to YAML mode.
2) Paste the contents of `dashboard.yaml`.
3) Save.

## Entity IDs
If any card shows "Entity not found", open Settings -> Devices & Services -> Entities
and replace the entity_id in `dashboard.yaml` with the one from your system.
